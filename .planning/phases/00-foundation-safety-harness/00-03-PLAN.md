## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| DPAPI file (.store/*.xml) → session | Restored credential crosses from disk to memory; wrong-user/machine or empty restore must not masquerade as a rights problem. |
| logged-in admin → AD | Rights are probed and acted on; a rights probe that performs a real write would itself be an unaudited mutation. |
| domain → session flags | Protected/deny resolution feeds every SAFE-05/06 refusal; a wrong SID hard-codes a bypass. |

(MENU-05/CONF-06) |
