Find all .psd1, .psm1, .ps1 files recursively under that module root, excluding paths whose FullName matches the regex \(tests|\.github|\.githooks)\. For each file call Set-AuthenticodeSignature -FilePath $file.FullName -Certificate $Certificate -HashAlgorithm SHA256; capture the returned Signature and assert Status -eq 'Valid'; if any signature is not 'Valid', throw "Signing failed for $($file.FullName): $($result.StatusMessage)". Include comment-based help with .SYNOPSIS, .DESCRIPTION, .PARAMETER for each parameter, and .EXAMPLE showing self-signed cert usage.
