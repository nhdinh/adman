## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-02-01 | Information Disclosure | Find verbs returning out-of-scope objects | high | mitigate | Loop -SearchBase per ManagedOUs root and re-check every emitted object with Test-AdmanInManagedScope (D-02). |
| T-02-02 | Information Disclosure | Read path wrongly hiding protected accounts | medium | mitigate | Apply ONLY Test-AdmanInManagedScope on reads; deny-list and protected-account checks remain mutation-only (D-02, RESEARCH §Pitfall 7). |
| T-02-03 | Tampering | LDAP filter injection via user input | medium | mitigate | Escape filter literals with Escape-AdmanLdapFilterValue before interpolating into -Filter strings. |
| T-02-SC | Tampering | Package installs | high | accept | No packages introduced this phase. |
</threat_model>
