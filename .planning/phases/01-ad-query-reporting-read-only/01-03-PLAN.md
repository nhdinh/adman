    Extend tests/Mocks/ActiveDirectory.psm1 so Search-ADAccount accepts -AccountDisabled, -AccountExpired, -LockedOut, -PasswordExpired, -SearchBase, -SearchScope, -ResultPageSize, -Server, -UsersOnly, and -ComputersOnly. Return AdmanMock-tagged objects with appropriate state flags.
