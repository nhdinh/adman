Ship the six AD user lifecycle Public verbs (USER-02..06). Each verb is a thin prompt-and-dispatch that validates init, builds the parameter hashtable, and calls Invoke-AdmanMutation. All gate machinery (create path, password plumbing, PDCe override, move destination validation) was built in Plan 02-01; this plan is purely the Public/ + tests/ layer.

Purpose: USER-02..06 are the highest-value single-object writes. They exercise every piece of the Phase 2 gate infrastructure (D-01 create, D-05 password, PDCe unlock, move validation).
Output: Six Public verbs + five test files. No Private/ changes.
</objective>

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| operator -> Public verb | The operator supplies Identity/Name/sAMAccountName/UPN/ParentOuDn/TargetPath and optionally a SecureString password |
| Public verb -> gate | The verb builds $Parameters and calls Invoke-AdmanMutation; the gate enforces all policy |
| password source -> SecureString | CSPRNG output or Read-Host -AsSecureString; never marshaled to plaintext |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-02-01 | Tampering | New-AdmanUser sAMAccountName | medium | mitigate | Length validated <= 20 before the gate call; uniqueness pre-flight in the gate refuses collision (D-01, Pitfall 6) |
| T-02-02 | Information Disclosure | Set-AdmanUserPassword / New-AdmanUser password handling | critical | mitigate | SecureString born in New-AdmanRandomPassword / Read-Host -AsSecureString; passed ONLY into the gate's $Parameters; never marshaled to plaintext; audit writer never receives $Parameters (D-05) |
| T-02-05 | Elevation of Privilege | Unlock-AdmanUser against non-PDCe DC | medium | mitigate | -Server pinned to (Get-ADDomain).PDCEmulator; LockedOut read first on the PDCe; no-op with clear message when not locked (Pitfall 2) |
| T-02-08 | Elevation of Privilege | Move-AdmanUser -TargetPath | high | mitigate | Destination OU validated under managed roots BEFORE the gate call (component-boundary anchored via ConvertTo-AdmanNormalizedDn); throws precise out-of-scope message (Pitfall 7) |
</threat_model>
